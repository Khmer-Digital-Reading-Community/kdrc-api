# ⚙️ KDRC API (តោះអានសៀវភៅ)

KDRC API is the backend service powering the Khmer Digital Reading Community platform.  
It handles authentication, book management, content delivery, and community interactions.

## ✨ Features
- Authentication & authorization (JWT)
- Book and chapter management
- Story publishing system
- Comments and ratings
- Book exchange system
- Payment & subscription handling
- Notifications system (Optional)

## 🛠 Tech Stack
- NestJS
- PostgreSQL
- Prisma / TypeORM
- JWT Authentication
- RESTful API

## 🎯 Goal
Provide a scalable, secure, and efficient backend to support a growing reading and writing community.
